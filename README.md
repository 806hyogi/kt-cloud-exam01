# 🐱 고양이 달리기 프로젝트

![exam01](https://github.com/user-attachments/assets/ad334cb9-aa6b-4197-9a59-97c3d5fa715a)

## 프로젝트 소개
간단한 React 프로젝트로, 사용자가 선택한 고양이 수만큼 고양이가 트랙에서 달리는 시뮬레이션 게임입니다.  
- 고양이들의 달리기 속도는 랜덤
- 완주 순위 표시
- 1등 고양이 커스텀 다이얼로그 표시

---

## 주요 기능
- 🏁 고양이 달리기 애니메이션  
- 🥇 완주 순위 실시간 표시  
- ⏱ 랜덤 달리기 속도  
- 🔄 새로고침 및 재시작 기능  

---

## 기술 스택
- React  
- React Router DOM  
- CSS 애니메이션
